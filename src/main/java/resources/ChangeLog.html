Samurai Graph 2.0.0

Released at 2010-11-23.
  • Added the netCDF files as the input file format.
  • Added the netCDF file to the file format of data set.
  • Enabled to switch the orientation of the bar graph and error bars between horizontal and vertical.
  • Enabled to shift the location of the line graph and the bar graph.
  • Enabled to switch whether to move or not the symbols such as the shape objects as axis range is changed.
  • Enabled to change the angle of the tick labels and the scale numbers.
  • Added functions to paint the background of the shape objects in a simple color, a pattern or gradation with transparency.
  • Added pseudo-color map to the graph type.
  • Revised the wizard dialogs for input data.
  • Added functions to operate the application from command-line input.
  • Improved scalar-type XY data significantly.
  • Revised commands of the paper size in the menu bar.
  • Added commands to change the front-back order of the objects.
  • Added commands to change the up-down order of the legend.
  • Fixed to show error bars above and below each data point ignoring the sign of error values.
  • Enabled to input the file name by drag-and-drop onto the text field on a dialog.
  • Enabled to zoom in or out the graph by holding down the control key and scrolling the mouse wheel.
  • Improved the grid lines for log-scaled axes.
  • Fixed cumulative minor bugs.

Samurai Graph 0.2.0

Released at 2004-11-10.
  • Arranged menu entries in the menubar.
    • Hid temporary entry for not implemented features.
    • Added new entries for some features below:
      • Insert Symbol (label, significant difference, line-break, timing line)
      • Zooming
      • Grid Properties
      • Version upgrade
      • Change Logs
    • Remapped short cut keys of major functions.
  • Arranged menu entries in the popup menu of some objects.
    • Added property entry for all popup menus.
  • Added version upgrade feature.
    • It can to check latest version via network at start up time. If newer version is available, information about it is displayed.
      • On Windows, it can automatically upgrade to latest version.
      • On other platforms, it can download latest version.
    • Added menu entry [Help]-[Upgrade] to menubar.
    • property dialog
      • Frequency setting for automatically latest version checking
      • Manually upgrade button
  • Other new features and changes
    • Added splash window.
    • Hid vector type data entry from data type selection dialog.
    • Added automatically zoom out to fit a window function. It can to use by menu entry [Layout]-[Zoom]-[Auto Zoom].
    • Added toggle switch of the grid line visible status. It can to switch from menu entry [Layout]-[Grid]-[Visible] or window property dialog.
    • Added recovery sequence for symbol insert failure. If you fail symbol inserting, the message dialog is shown and you can continue insert operation.
    • Changed symbol superposition policies. Superposition of same type symbol (label, significant difference, break-line, timing line) can to change by popup menu.

    • Added popup menu entry for axis and legend visible control.
    • Changed show action of axis property dialog at axis invisible. It can to show by double click operation of left button of mouse on rectangle border of the figure.
    • Added change logs dialog. It can to show by menubar entry [Help]-[Change Log].
